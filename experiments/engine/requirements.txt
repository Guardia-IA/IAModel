#   1) PyTorch con CUDA (según tu versión de CUDA):

#
#   2) Dependencias del paquete engine:
#      pip install -r engine/requirements.txt
#
#   3) TensorRT (si no está en el sistema):
#      - Ubuntu: sudo apt install nvidia-tensorrt  (o desde .deb de NVIDIA)
#      - Pip (cuando hay wheel para tu plataforma):
#        pip install nvidia-tensorrt
#
# Uso:
#   python engine/build_engines.py
#   python engine/build_engines.py --model yolo11x-pose.pt
